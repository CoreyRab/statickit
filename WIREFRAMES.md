## Key UX Principles Applied

### 1. Respect Original Aspect Ratio
- **Auto-detect** the uploaded image's aspect ratio
- **Display prominently** so user knows what to expect
- **Generate in same ratio** by default - no surprises
- **Additional sizes are optional** and clearly cost extra

### 2. Clear Credit System
- Show credit cost at every decision point
- 1 credit = 1 image (regardless of size)
- Additional sizes multiply the cost transparently
- Always show remaining credits

### 3. Progressive Disclosure
- Start simple: upload â†’ analyze â†’ pick variations â†’ generate
- Advanced options (more sizes) come AFTER initial generation
- Don't overwhelm with size options upfront

### 4. Platform-Aware
- Show common platform names alongside dimensions
- Offer "Meta preset" / "Google preset" for quick multi-size export
- Help users understand which size goes where
